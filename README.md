# VC2 (Virtual Classroom Clicker) #
# README #

This README details the steps necessary to get this application up and running.

### Requirements ###

* Node JS
* Gulp

### How to run ###

1. Open the terminal and navigate to vc2 directory
2. `npm install`
3. `npm start`
4. Go to http://localhost:3000 in a browser

### To build and run ###

1. Open the terminal and navigate to vc2 directory
2. `npm install`
3. `gulp`
4. Go to http://localhost:3000 in a browser
